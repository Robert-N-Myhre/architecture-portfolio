# How to Update My Architecture Portfolio

1. Edit Markdown files or mkdocs.yml as needed.
2. Save changes.
3. Commit and push changes to main:
4. GitHub Actions deploys changes
   check Actions for progress.status
5. (Deprecated) Deploy to GitHub Pages with MkDocs:
   mkdocs gh-deploy
6. Wait ~1 minute and check: https://robert-n-myhre.github.io/architecture-portfolio/
